# voting-machine

Code for Voting Machine used at Latcraft meetups (<http://latcraft.lv>) and DevTernity conference (<http://devternity.com>).

## Installation

Use latest Raspbian distribution.

    gradlew init -Phosts=192.169.0.16 -Ptype=latcraft-voter
